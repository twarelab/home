testX JIG

testX JIG란

티웨어랩의 testX JIG는 다품종 소량 생산 PCBA 제품의 테스트를 위하여 최적화된 범용 JIG 장치 입니다.
제품별로 JIG를 따로 제작할 필요 없이, 범용 JIG에 제품별 Fixutre 박스만 교환하여 다양한 제품을 테스트 할 수 있습니다.

지그 특장점

비용 절감

  • JIG 재사용
  • 제품별 전용 Fixture 박스만 제작

높은 생산성

  • 최대 10개 제품 동시 테스트
  • 손쉽고 유연한 JIG 개수 확장

QR 코드 이력 관리

  • 중복 테스트 차단
  • 테스트 누락 차단
  • 테스트 이력 DB 등록

주요 기능

  • 다중 QRcode 인식 가능 : 테스트 하는 제품의 QRcode를 동시에 인식
  • 펌웨어 프로그래밍 및 기능 검증
  • 외부 인터페이스 포트에 대한 동작 검증
  • Peripheral 테스트 기능 : GPIO/ADC/UART/SPI/I2C/Ethernet
  • 테스트 로그 저장 기능 : 테스트 단계별 로그 데이터를 클라우드(DB sever)에 저장
  • 소비 전류 측정
  • 그 외 검증이 필요한 항목에 대한 테스트 수행

제품 구성

  1. 1카메라부
    • 테스트 제품 이력 추적을 위한 QR 코드 인식
  2. 2Fixture부
    • 테스트 제품을 접촉하는 부분
    • Fixture를 손쉽게 교환 가능
    • 제품별로 전용 Fixture 필요
  3. 3통신제어부
    (제품 후면)
    • 테스트 절차를 수행하고 테스트 단계별 상태를 화면에 표시하는 제어보드(Embedded Linux)
    • 개별 제품에 대한 직접적인 테스트를 수행하는 최대 10개의 채널보드(MCU 보드)
    • 채널보드는 다양한 통신 인터페이스 지원

범용 JIG 1대와 여러 개의 Fixture box를 조합하여

Portfolio

센서 데이터 수집 시스템

  • 온도, 습도, 중량, 진동 등 다양한 센서에서 발생한 이벤트 및 데이터를 실시간으로 서비스 서버로 전송
  • 시스템 성능에 영향없이 수집된 센서 데이터, 고객 맞춤형 보정 데이터 동시 백업

센서 이벤트 수신

  • 실시간으로 시리얼을 통한 센서 데이터 및 이벤트 수신
    예) 20ms 마다 센서 데이터 수집, 조건은 만족하는 경우 센서 데이터 수집 등
  • 이벤트 트리거 설정 기능
  • 센서 데이터 보정 기능

센서 데이터 관리

  • 고객 맞춤형 데이터 기능
  • 센서 데이터를 TCP/IP 통신 네트워크를 이용한 서버 전송
  • 로컬 DB 백업 기능 (데이터 이중화)

Specification

개발 환경

라즈베리파이, 센서 데이터 보드, 시리얼 통신 보드

개발 언어

Python, SQLite

지원 운영체제

Raspbian Linux

주요 특징
  • 실시간 센서 데이터 및 이벤트 수집
  • 고객 맞춤형 데이터 보정 – AI, 고객 알고리즘
  • Serial 통신 program
  • 센서 데이터 보드 제어 program
  • Multi-Thread program
  • TCP/IP Program (HTTP, HTTPS, REST API)
  • 백업 로컬 DB 시스템

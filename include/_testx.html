테스트 시스템 도입

시스템도입 모델은 EMS 기업에게 적합한 방식으로 testX서비스를 위한 테스트 라인을 EMS 기업에 구축하고 직접 테스트를 수행합니다.JIG, JIG SW, Manager SW, DB Server, 웹서비스는 티웨어랩의 솔루션을 사용합니다.

테스트 외주서비스

외주 서비스 모델은 별도의 테스트 라인 구축없이 티웨어랩에 구축된 테스트 라인을 사용하는 방식입니다. 테스트할 제품을 티웨어랩에 전달하고, 이후 모든 테스트 과정을 티웨어랩에서 수행하는 모델입니다.
